## 1. What is Docker?

## Docker is a platform used to run applications inside containers.

A container includes:

* application code
* dependencies
* libraries
* runtime environment

Everything runs in an isolated environment.
--------------------------------------------------------------------------------
##  2. Problem Docker Solves

## A very common issue in development:

“It works on my machine.”

### But on another system:

* dependencies may differ
* versions may mismatch
* environment may not be the same

### Docker solves this by providing:

* consistent environments
* portability
* reliable deployments
